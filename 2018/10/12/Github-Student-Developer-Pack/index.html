Github Student Developer Pack

Posted on 2018-10-12
Github提供的免费开发者福利。

计算机相关专业的学生都是知道Github是个好东西，但是它提供的免费开发者福利确实鲜有人知。

获取学生包

首先点击地址 https://education.github.com/pack/

点击Get your pack.跳转之后根据提示该登陆登陆，该验证学生邮箱验证。然后再回来看看就发现自己有不少好东西进账了。

说实话一开始不认识几家公司…不妨碍，有时间挨个摸索一下。好了，现在讲点有意思的东西（敲黑板）。

使用DigitalOcean云服务

DigitalOcean简介

DigitalOcean（下面简称DO） 是一家提供云服务的公司，但是其坐标在国外，所以聪明的小伙伴已经知道可以用来干什么了。这里github学生包给了50刀的代金券，最便宜的服务器5刀一个月也可以免费用10个月了。点击Get my code可以得到自己的代金券码，复制下来，然后点开DO的官网。

注册登陆

然后就是简单的操作注册登陆操作了，值得注意的是，要是用DO的服务，必须要证明自己拥有付款能力，要不使用Paypal要不拥有信用卡。前者需要先充值5刀进账户，后者会先扣1刀，证明可以使用的话会退回来（没尝试过，有偏差不负责）。即便先充5刀也很划算。（ps DO登陆的时候很考虑安全，每次都会给邮箱发验证码，登陆时需要输入。）

创建服务器

下面就得创建自己的服务器：点击create-droplets

然后选择系统版本和规格：我选的是Ubuntu，考虑到用的人比较多出了问题可能容易找到解决方案，然后规格就选了最便宜的，毕竟不需要很高性能。

然后是节点，可以根据选择喜好，我选的是San Franciosco。

然后可以可以添加一个自己电脑的SSH keys，方便访问服务器，不添加的会通过邮箱发送默认账户和密码（也是听说，没尝试过），手输登陆也还行

登陆服务器

好了，然后登陆服务器吧，我用的是mac，所以终端SSH访问了

具体的访问喜好应人而异了，这里就不展开讲了，有需求的同学可以自行搜索了。
